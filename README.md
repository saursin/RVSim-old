# RVSim
A simple RISC-V ISA simulator

## Build Instructions
1. Create a build directory
    ```bash
    $ mkdir build
    $ cd build
    ```

2. Run CMAKE to generate Makefile
    ```bash
    $ cmake ..
    ```

3. Run Make to build RVSim
    ```bash
    $ make
    ```

4. Run the executable
    ```bash
    $ ./rvsim --help
    ```
